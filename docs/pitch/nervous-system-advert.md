## Spread 6 — The four nerves we install first

> ## "They notice. They push. They escalate. They close."

| | |
|---|---|
| ![Receiving](images/04-receiving.png) | **Receiving & Match** — photographs delivery notes, flags any line that doesn't match the open PO. Won't drop a short delivery until it's reconciled. |
| ![Transfer](images/05-transfer.png) | **Inter-Building Transfer** — watches every component move between bays. Flags dispatched-but-never-arrived in 30 minutes. Even a hundred-metre walk now lives in the system. |
| ![Exception triage](images/08-exception-triage.png) | **Production Exception Triage** — every blocker on the floor has an owner, an age, and a status. Old blockers escalate. None rot. |
| ![Daily brief](images/06-daily-brief.png) | **Daily Control-Tower Brief** — one Telegram message every morning. Late POs, stuck transfers, blocked jobs, payroll anomalies. Read in five seconds. |
